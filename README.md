# Currículo Online - Gabriel Kohler Vellar

## 📋 Sobre o Projeto

Este projeto consiste em um currículo online desenvolvido com o objetivo de apresentar minhas competências técnicas, formação acadêmica e experiências profissionais de forma moderna e interativa.

O site foi desenvolvido como parte do meu processo de atualização e evolução profissional na área de Tecnologia da Informação, unindo conhecimentos adquiridos ao longo da minha trajetória acadêmica e profissional.
